# Evaluation - Read Overlapping (for Rawsamble)

We assume your current directory points to this directory ([`rawsamble`](./))

## Organization

Each directory includes a single-liner script to 1) run Rawsamble and minimap2 for the corresponding dataset where the dataset name is specified in the directory name and 2) generate the results.

## Running and comparing the tools

### d2_ecoli_r94 (Annotated as D1 in our Rawsamble manuscript)

Please read the [README](./d2_ecoli_r94/README.md) file to run and compare the tools for this dataset.

To start running each tool, enter [d2_ecoli_r94](./d2_ecoli_r94/)

```bash
cd ./d2_ecoli_r94
```
